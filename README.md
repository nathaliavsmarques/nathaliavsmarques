## 📁 Projetos em Destaque

### 🦾 Controle de Robô Quadrúpede com PPO – Ant-v5 (MuJoCo) 
Implementação do algoritmo **PPO do zero** com arquitetura Actor-Critic em 
PyTorch para controle de robô quadrúpede 3D em ambiente MuJoCo.

- Espaço de estados com **105 dimensões** e espaço de ações contínuo de 
**8 dimensões** (torques das juntas do robô)
- Aplicação de **Clipping de razão** para estabilização do treinamento e 
prevenção de colapso de política
- Agente convergiu de recompensa **-3.000 → +600** após 3 milhões de 
timesteps, completando episódios inteiros sem cair

`Python` `PyTorch` `PPO` `Actor-Critic` `Gymnasium` `MuJoCo` `Google Colab`
